# Клеми, клемники та штекери в Eplan: теоретична частина 

## 7.1. Звіти

в розробці

------

клеми

| Властивість                                                  | Опис                                                         |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| Об’єднати багаторівневі клеми <13052>                        | Визначає, чи відображати багаторівневі клеми поруч або одну під одною. Актуально для схем клемників. |
| Відображати виводи пристрою <13113>                          | Якщо властивість активована, у специфікаціях клемників і схемах підключення для кожної пари виводів пристрою генерується один рядок. Клеми з більш ніж двома виводами автоматично розподіляються на кілька рядків. Додатково виводяться також непідключені виводи клем. |
| Перемички: радіус для вузлів <13014>                         | Якщо перемички у специфікаціях клемників відображаються графічно, на основній лінії для кожної «ніжки» перемички ставиться невелике заповнене коло. Тут задається його радіус (у програмних координатах). |
| Перемички: висота для зовнішньої дротової перемички <13016>  | Висота першого шару зовнішньої дротової перемички у специфікації клемника. Якщо не задано, використовується властивість «Перемички: висота першого шару». |
| Перемички: висота для внутрішньої дротової перемички <13015> | Висота першого шару внутрішньої дротової перемички у специфікації клемника. Якщо не задано, використовується властивість «Перемички: висота першого шару». |
| Перемички: висота першого шару <13010>                       | Висота перемички першого шару у специфікації клемника.       |
| Перемички: інтервал <13012>                                  | Інтервал між двома перемичками сусідніх шарів.               |
| Перемички: інтервал для зовнішньої дротової перемички <13018> | Інтервал між двома зовнішніми дротовими перемичками сусідніх шарів. Якщо не задано, використовується властивість «Перемички: висота першого шару». |
| Перемички: інтервал для внутрішньої дротової перемички <13017> | Інтервал між двома внутрішніми дротовими перемичками сусідніх шарів. Якщо не задано, використовується властивість «Перемички: висота першого шару». |
| Перемички: радіус виводів пристрою <13013>                   | У специфікаціях клемників вузлові точки перемичок відображаються як невелике заповнене коло. Тут задається його радіус (у програмних координатах). Початкові та кінцеві «ніжки» не отримують точки. |
| Перемички: товщина лінії <13028>                             | Товщина лінії для відображення перемички.                    |
| Перемички: кількість шарів <13029>                           | Загальна кількість шарів перемички.                          |
| Представлення перемичок <13027>                              | Графічне відображення перемичок: 0 – лінія, 1 – кут.         |
