# Markdown

**Про що:**

- про використання MarkDown як мови розмітки для технічної документації, статичних Веб-сайтів, супровідних текстових описів, тощо
- про базовий та розширений синтаксис MarkDown
- про практичне створення та редагування MD-документів у різних редакторах
- про роботу з формулами LaTeX та діаграмами Mermaid у Markdown

**Що отримаєте:**

- знання принципів роботи MarkDown та областей його застосування
- розуміння базового і розширеного синтаксису MD
- вміння створювати структуровані документи у форматі Markdown
- навики роботи з редакторами MarkDown (Notepad++, Typora)
- навики використання формул LaTeX і діаграм Mermaid у документації

**Що потрібно знати вже:**

- базові навики роботи з файловою системою операційної системи
- елементарні навики роботи з текстовими редакторами
- загальне розуміння структури документів

**Необхідні інструменти:**

- текстовий редактор (Notepad++, Typora або аналогічний)
- плагін Markdown Panel для Notepad++
- операційна система Windows / Linux / macOS
- веб-браузер для перегляду документації

**Джерела:** 

- https://www.markdownguide.org/
- https://typora.io/
- https://mermaid.js.org/
