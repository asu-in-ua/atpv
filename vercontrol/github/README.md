# Основи GitHub

**Про що:**

- про GitHub як платформу для зберігання, публікації та спільної роботи з матеріалами;
- про роботу з репозиторіями через веб-інтерфейс без використання локальних інструментів;
- про обговорення, історію змін та публікацію веб-сторінок на базі репозиторію.

**Що отримаєте:**

- знання можливостей GitHub як платформи, незалежно від локального Git;
- знання типів репозиторіїв та моделей доступу;
- знання призначення сервісів Code, Issues, GitHub Pages; 
- вміння
  - орієнтуватися в інтерфейсі GitHub;
  - працювати з файлами репозиторію через браузер;
  - використовувати історію змін і обговорення;
  - публікувати матеріали у вигляді веб-сторінок;
- навики
  - практичної спільної роботи з матеріалами;
  - використання GitHub як єдиного джерела і публічного представлення контенту.

**Що потрібно знати вже:**

- базові навички роботи з комп’ютером;
- роботу з веб-браузером;
- загальне уявлення про файли та каталоги.

**Необхідні інструменти:**

- персональний комп’ютер;
- веб-браузер;
- доступ до мережі Інтернет.

**Джерела:** 

- офіційна документація GitHub;
