# Від локального Git до GitHub

**Про що:**

- поєднання локальної роботи з репозиторіями Git та віддаленими репозиторіями GitHub, принципи синхронізації змін і базові сценарії спільної роботи.

**Що отримаєте:**

- розуміння ролі GitHub як віддаленого репозиторію;
- навички зв’язування локального Git-репозиторію з GitHub;
- практику обміну змінами через push і pull;
- спостереження та аналіз ситуацій виникнення конфліктів без їх розв’язання.

**Що потрібно знати вже:**

- базові поняття та навики роботи з Git в межах мікромодуля [Основи систем керування версіями та Git](../git/README.md);
- базові поняття та навики роботи з GitHub в межах мікромодуля [Основи GitHub](../github/README.md);

**Необхідні інструменти:**

- Git (локальна інсталяція);
- обліковий запис GitHub;
- веб-браузер;
- Git Bash або Git GUI.

**Джерела:** 

- офіційна документація Git;
- офіційна документація GitHub.
