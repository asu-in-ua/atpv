# Сигнали та їх типи в системах керування

**Про що:**

- про базові поняття сигналу як фізичного носія інформації в системах керування, 
- про фундаментальні типи сигналів та способи кодування інформації в них, без прив’язки до конкретних платформ, протоколів або реалізацій.

**Що отримаєте:**

- розумітимете, що таке сигнал у контексті систем керування
- зможете відрізняти аналогові та дискретні сигнали на концептуальному рівні
- розумітимете, чому PWM не є окремим типом сигналу
- знатимете, що таке модуляція і яку роль вона відіграє в автоматизації
- отримаєте основу для подальшого вивчення датчиків, PLC, мікрокомп’ютерів та виконавчих механізмів

**Що потрібно знати вже:**

- базові уявлення про фізичні величини (напруга, струм)
- поняття логічних станів 0 та 1
- загальне розуміння, що інформація в технічних системах передається сигналами
