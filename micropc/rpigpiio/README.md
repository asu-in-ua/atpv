# Raspberry PI GPIO

**Про що:**

- про апаратні та програмні основи роботи з GPIO Raspberry Pi
- про підключення простих дискретних навантажень (світлодіод)
- про обмеження GPIO за струмом і напругою та безпечні схеми підключення
- про керування GPIO з командного рядка та з Node-RED

**Що отримаєте:**

- знання
  - принципів роботи GPIO як виходів напруги
  - правил безпечного підключення навантажень до Raspberry Pi
  - ролі обмежувального резистора та транзистора в схемах керування
  - основ використання PWM для керування яскравістю
- вміння
  - читати схеми підключення GPIO
  - розраховувати струм у простих колах зі світлодіодом
  - збирати схему на макетній платі
  - керувати GPIO за допомогою `pinctrl`
  - створювати прості потоки керування GPIO в Node-RED
- навики
  - практичної роботи з Raspberry Pi GPIO
  - налагодження апаратно-програмної взаємодії
  - використання Node-RED для керування фізичними входами/виходами

**Що потрібно знати вже:**

- базові поняття електрики (напруга, струм, опір), закон Ома
- елементарні навички роботи з командним рядком 
- базове уявлення про Raspberry Pi та його призначення в межах матеріалу [Знайомство з Raspberry PI та встановлення на неї ОС](../rpiinstall/README.md)

**Необхідні інструменти:**

- робочий ПК (з віконною ОС)
- Raspberry Pi (з ОС Raspberry Pi OS)
- макетна плата, світлодіод, резистор, з’єднувальні проводи
