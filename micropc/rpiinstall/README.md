# Знайомство з Raspberry PI та встановлення на неї ОС

**Про що:**

- про Raspberry Pi як одноплатний комп’ютер
- про встановлення та початкове налаштування операційної системи
- про роботу з Raspberry Pi як мережним пристроєм

**Що отримаєте:**

- загальне уявлення про Raspberry Pi та його місце серед обчислювальних платформ
- навички встановлення операційної системи на Raspberry Pi без підключення монітора, клавіатури та миші
- розуміння базових підходів до віддаленої роботи з Raspberry Pi через мережу
- практичний досвід використання утиліт для віддаленого доступу та обміну файлами

**Що потрібно знати вже:**

- основи роботи з персональним комп’ютером
- базові поняття про операційні системи та комп’ютерні мережі

**Необхідні інструменти:**

- Raspberry Pi
- карта пам’яті microSD
- блок живлення
- Wi-Fi маршрутизатор або локальна мережа

**Джерела:** 

- За основу взяті матеріалу курсу [Програмна інженерія в системах управління](https://github.com/pupenasan/ProgIngContrSystems)
