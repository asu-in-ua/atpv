# Узгодження електроніки з керуючими платами

**Про що:**

- про електричну взаємодію керуючих плат із зовнішніми компонентами
- про обмеження входів і виходів (напруга, струм, режими роботи)
- про узгодження рівнів сигналів між платою і макетною електронікою
- про типові схеми підключення на макетній платі без пошкодження плати
- про практичні аспекти interfacing у прототипах

**Що отримаєте:**

- знання
  - електричних характеристик GPIO та аналогічних пінів керуючих плат
  - різниці між входами і виходами з електричної точки зору
  - причин використання резисторів, транзисторів, буферів
  - типових помилок при прямому підключенні компонентів
- вміння
  - безпечно підключати світлодіоди, кнопки, датчики
  - узгоджувати рівні 3.3 В і 5 В
  - оцінювати, чи можна підключати компонент напряму
  - читати прості схеми підключення з документації
- навики
  - збирання прототипів на макетній платі
  - роботи з GPIO без ризику пошкодження плати
  - швидкого аналізу електричної сумісності компонентів

**Що потрібно знати вже:**

- базові поняття про напругу, струм, опір
- [дискретні та цифрові сигнали](../../signals/basicsign/README.md)
- призначення GPIO або пінів мікроконтролера
- роботу з макетною платою на базовому рівні

**Необхідні інструменти:**

- керуюча плата (Raspberry Pi або мікроконтролер)
- макетна плата і з’єднувальні дроти
- базовий набір компонентів (резистори, LED, кнопки)
- джерело живлення для прототипу
- мультиметр (бажано, але не обов’язков

**Джерела:** 

- Molloy, Derek. *Exploring Raspberry Pi: Interfacing to the Real World*. Chichester: Wiley, 2016. 
- Gay, Warren. *Raspberry Pi Hardware Reference*. New York: Apress, 2014.
